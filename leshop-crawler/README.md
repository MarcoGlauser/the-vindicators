# How to run crawler

* create secrets.py
* Open http://leshop.ch in a browser
* Copy the header "leshopch" from a request
* Define the variable `leshop_token = 'TOKEN'` in secrets.py
* Define the variable `wolfram_app_id = 'APP_ID'` in secrets.py
